Phoby

VR exposure therapy for phobias

2019

Phoby is a virtual reality application designed to help users confront and manage common phobias through controlled exposure therapy. The experience places users in gradually intensifying scenarios tailored to specific fears—heights, spiders, public speaking—while maintaining a sense of safety and control.

The project explored how immersive environments could replicate the physiological responses associated with phobias while giving users tools to self-regulate. Built in Unity for Oculus Quest, the experience included biometric feedback integration and therapist-guided session modes.

key features

  • Graduated exposure levels with user-controlled intensity
  • Multiple phobia modules (acrophobia, arachnophobia, glossophobia)
  • Real-time heart rate monitoring via Bluetooth integration
  • Session logging for therapeutic review

technical details

Built with Unity 2019.4 LTS targeting Oculus Quest. Implemented custom shader work for environmental atmosphere and used Photon for optional remote therapist observation mode. Heart rate data pulled from Polar H10 chest strap via BLE.
